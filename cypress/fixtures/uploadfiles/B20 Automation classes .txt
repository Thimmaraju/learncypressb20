xpath 

xml path

2 types 

1. Absolute xpath
    
	starts from root node 
	
	always starts with /
	
	  Ex: 
	  
/html/body/div/div[1]/div/div[1]/div/div[2]/div[2]/form/div[1]/div/div[2]/input

2. Relative xpath 
    starts from relative node 
	
	always starts with //
	
	//input[@name="username"]
	
	cy.xpath('//input[@name="username"]')
	
	
1. Basic Syntax 

  //tagname[@attributename='attributevalue']
  
  
  //input[@type="password"]
  
2. Text

  //tagname[text()='fulltext']
  
  //p[text()="Forgot your password? "]
  
  cy.xpath('//p[text()="Forgot your password? "]').click()

   //tagname[contains(.,"partial text")]
   
   //p[contains(.,"Forgot your")]
   
   //p[contains(., "With Cypress, you can easily")]
   
   
4. AND 

  Both atrributes should be present in the node 

   //tagname[@attr1='attr1value'][@attr2='attr2value']

    //tagname[@attr1='attr1value'and @attr2='attr2value']
	
	   //input[@name="username"and @placeholder="Username"]
   
   
5. OR 

  Any one attribute should be present 

  //tagname[@attr1='attr1value'or @attr2='attr2value']
  
    //input[@name="username"or @placeholder="Raju"]
	
6. Prefix 

   id=message12335723
   
   id=message65986895
   
   //tagname[starts-with(@attrname, "prefixvalue")]
